Features

Everything developers need

Elara packs powerful features into a clean interface, letting you focus on what matters: building.

💬

Smart Chat Interface

Full-featured chat with streaming, file uploads, web search, reasoning mode, and conversation history per tab.

⚡

Multi-Provider Support

Access 9+ AI providers through one interface. Switch between Claude, DeepSeek, Groq, and others instantly.

👥

Multi-Account Management

Manage multiple accounts per provider. Perfect for separating personal and work usage or team collaboration.

📊

Usage Analytics

Track token consumption, request counts, favorite models, peak usage times, and costs across all accounts.

✍️

Git Commit Generator

Connect to Git, stage your changes, and let Elara write conventional commit messages in one click.

🖥️

CLI + GUI Combined

Use Elara from the desktop or your terminal. The app runs a local API server at localhost:8888 for CLI tools.

📑

Tabbed Conversations

Work on multiple topics simultaneously. Each tab has its own model, provider, settings, and full chat history.

⌨️

Global Hotkeys & Tray

Access Elara instantly from anywhere with configurable keyboard shortcuts. Runs silently in the system tray.

🔒

Local-First Privacy

API keys and data stay on your machine. Elara connects directly to AI providers — no middleman servers.

⚡ Quick Start

  1. Download and install Elara for your platform
  2. Open Elara and go to Accounts → Add Account
  3. Enter your API key or login credentials for any AI provider
  4. Open Playground, select your provider and model, and start chatting
  5. Optional: Enable Git integration in Settings for commit message generation
All releases are available on GitHub Releases. Found a bug? Open an issue.

Privacy Policy

Last updated: May 1, 2025

TL;DR: Elara is local-first software. Your API keys, credentials, and conversation data are stored on your device. Elara itself does not collect, transmit, or store any of your personal data on its servers.

1. Who We Are

Elara is developed and maintained by KhanhRomVN, an independent developer. You can contact us at khanhromvn@gmail.com.

2. What Data We Collect

Elara does not collect any personally identifiable information. Specifically:

  • We do not collect your name, email address, or any account information.
  • We do not collect your API keys or AI provider credentials.
  • We do not collect your conversation history or prompts.
  • We do not use analytics SDKs, tracking pixels, or telemetry tools.

3. Data Stored Locally

The following data is stored exclusively on your local device:

  • API keys and credentials — stored in your operating system's secure storage
  • Conversation history — stored in a local database on your machine
  • Application preferences and settings — stored in your user config directory
  • Usage statistics — stored locally; never transmitted to Elara servers

4. Third-Party AI Providers

When you interact with an AI model through Elara, your prompts and conversation data are sent directly to the AI provider you have configured (e.g., Anthropic, Groq, Mistral). Elara acts solely as a client application — it does not intercept, store, or log any of this communication.

Each AI provider has its own Privacy Policy and Terms of Service. You are responsible for reviewing and complying with those policies. Elara is not affiliated with any of these providers.

5. Automatic Updates

Elara may check for updates by making anonymous requests to our update server (GitHub Releases). These requests contain only the current version number and your operating system type. No personal data is included.

6. Open Source

Elara is open-source software. You can review the complete source code on GitHub to verify our privacy practices.

7. Children's Privacy

Elara is not directed at children under the age of 13. We do not knowingly collect information from children.

8. Changes to This Policy

We may update this Privacy Policy from time to time. Changes will be reflected by the "Last updated" date at the top of this page and noted in the Changelog.

9. Contact

Questions about this Privacy Policy? Contact us at khanhromvn@gmail.com or open an issue on GitHub.
